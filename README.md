# pinyougou-mall
基于SOA架构的分布式B2C2C商城
# 主要技术
前后分离，分布式SOA架构。<br/>
前端：angularJS + Bootstrap<br/>
后端：Spring+SpringMVC+MyBatis+SpringBoot<br/>
安全：SpringSecurity<br/>
数据库：MySQL+Redis<br/>
中间件：Dubbox,Zookeeper,ActiveMQ,MyCat<br/>
全文检索：Solr<br/>
Web服务器:Tomcat<br/>
代理服务器:Nginx<br/>
图片服务器:FastDFS<br/>
